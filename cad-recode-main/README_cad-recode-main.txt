📁 Φάκελος: cad-recode-main

Προέλευση: Το συγκεκριμένο repository περιέχει τον πηγαίο κώδικα για το εργαλείο CAD-Recode, ένα state-of-the-art ερευνητικό project που μετατρέπει νέφη σημείων (.ply) σε παραμετρικά μοντέλα CAD μέσω Python script (CadQuery) με χρήση Μεγάλου Γλωσσικού Μοντέλου (LLM).

🧠 Χρησιμοποιείται στο πλαίσιο της βιβλιογραφικής και πειραματικής βάσης του διδακτορικού ως πρότυπο μοντέλο για την παραγωγή CAD από point cloud.
