📁 Φάκελος: Clean Ply

Σκοπός: Περιέχει scripts για τον καθαρισμό νεφών σημείων (.ply) προτού χρησιμοποιηθούν για αναγνώριση ή ανακατασκευή εξαρτημάτων. Το preprocessing είναι κρίσιμο στάδιο στη ροή "scan → clean → segment → extract → model".

Πιθανές λειτουργίες:
- Downsampling (π.χ. voxel grid)
- Αφαίρεση θορύβου
- Cropping/ROI επιλογή
- Οπτική επαλήθευση
